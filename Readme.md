# AI | 夜白
一个专注于 AI 产品设计与实践的个人博客网站。
## 🌐 访问地址
- GitHub Pages: https://daihuayang.github.io

## 📝 项目介绍

这是我的个人博客，主要分享：
- AI 产品设计与实践经验
- LLM 应用开发思考
- 用户研究方法论
- AI 行业观察与趋势分析

## 🛠️ 技术栈

- **纯静态网站** - HTML5 + CSS3 + JavaScript
- **部署** - GitHub Pages
- **样式** - 现代化深色主题，响应式设计
- **动画** - CSS 动画 + Intersection Observer

## 📁 项目结构

```
├── index.html          # 博客主页（包含首页、博客、关于三个页面）
├── 404.html            # 404 错误页面
├── CNAME               # 自定义域名配置
├── .github/workflows/  # GitHub Actions 部署配置
└── README.md           # 项目说明
```
